# Git Basic Commands

## Initialize Repository
```bash
git init
```

## Check Status
```bash
git status
```

## Add Files
Add all files:
```bash
git add .
```

Add single file:
```bash
git add index.html
```

## Commit Changes
```bash
git commit -m "Index.html file inserted"
```

## View Commit History
```bash
git log
```

## Unstage File
```bash
git restore --staged index.html
```
